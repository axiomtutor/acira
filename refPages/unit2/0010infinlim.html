Lesson 0010: Infinite Limits

Let $(a_k)_{k=m}$ be a sequence of $\Bbb R$. If for every $\alpha\in\Bbb R$, there is a tail of the sequence which is bounded below by $\alpha$, then we write $$ \lim_{k\to\infty} a_k = \infty$$ If for every $\alpha\in\Bbb R$ there is a tail of the sequence which is bounded above by $\alpha$, then we write $$ \lim_{k\to\infty} a_k = -\infty$$ In either case, we say that the sequence has an infinite limit.
Throughout this section, assume that $(a_k)_{k=m}$ is a sequence of $\Bbb R$.
Theorem: Infin. lim. no lim.
If the limit is infinite, then the sequence has no limit.
Statement and proof. Statement: If $\lim_{k\to\infty}a_k=\pm\infty$ then $(a_k)_{k=m}$ has no limit.

Proof: Suppose $\lim_{k\to\infty} a_k = \infty$ and let $L\in\Bbb R$.
Then let the $p$th tail be bounded below by $L+1$.
We must then have that no tail can be within 1 of $L$, and therefore $\lim_{k\to\infty}a_k\ne L$.
Since $L$ was arbitrary, then no real number is the limit of $(a_k)_{k=m}$.
The proof is mutatis mutandis the same if $\lim_{k\to\infty}a_k=-\infty$.
$\Box$
Theorem: Infin. lim. zero recip.
If the limit is infinite the reciprocal limit is zero.
Statement and proof. Statement: If $\lim_{k\to\infty}a_k=\pm\infty$ then $\lim_{k\to\infty}\frac 1 {a_k} = 0$.
Proof: Suppose $\lim_{k\to\infty} a_k=\pm\infty$, and let $\varepsilon\in\Bbb R$.
Let the $p$th tail be bounded below by $1/\varepsilon$. Then for all $k\ge p$, we have $a_k \ge 1/\varepsilon$ and therefore $\frac 1 {a_k} \le \varepsilon$.
Therefore, $$\begin{aligned} \left|\frac 1 {a_k} - 0 \right| &= \frac 1 {a_k} \\ &< \varepsilon \end{aligned}$$ If $\lim_{k\to\infty}a_k = -\infty$ then by a simple proof, $\lim_{k\to\infty}(-a_k) = \infty$. From the above, then, $$ \begin{aligned} \lim_{k\to\infty} \frac{1}{a_k} &= -\lim_{k\to\infty} \frac{1}{-a_k} \\ &= -0 = 0 \end{aligned}$$ $\Box$
Theorem: Infin. lim. poly.
The limit of every nonconstant polynomial is infinite.
Statement and proof. Statement: Let $P(k)$ be a polynomial in $n$, with a positive leading coefficient. Then $\lim_{k\to\infty}P(k) = \infty$. If the leading coefficient is negative then $\lim_{k\to\infty} P(k) = -\infty$.

Proof: Let $P(k) = a_0+a_1k + \cdots a_nk^n = k^n(a_0/k^n + a_1/k^{n-1} + \cdots + a_n)$. Let $\alpha\in\Bbb R$.
For each $0\le i\le n$ there is some $k$ such that $a_i/k^{n-i} > \alpha$.
Theorem: Seq. rat. func.
Limit laws for rational functions.
Statement and proof. For any polynomial in $n$, $P(n)$, define $\text{deg}(P)$ to be the degree of $P$, which is the highest exponent of a nonzero term.
Statement: Let $P(n),Q(n)$ be two polynomials in $n$, and assume $Q(n)$ is not zero for $m\le n$.
Let $R(n) = \frac{P(n)}{Q(n)}$ be the rational function defined for $m\le n$.
  1. If $\text{deg}(P) < \text{deg}(Q)$ then $\lim_{k\to\infty}R(k) = 0$.
  2. If $\text{deg}(P) = \text{deg}(Q)$ then $\lim_{k\to\infty}R(k)$ is equal to the ratio of the leading coefficients of $P$ and $Q$.
  3. If $\text{deg}(P) > \text{deg}(Q)$ then $\lim_{k\to\infty}R(k)=\pm \infty$.


Proof:
If $\text{deg}(P) < \text{deg}(Q)$ then $\lim_{k\to\infty}R(k)=0$. Let $P(k) = a_0+a_1k+\cdots +a_pk^p$ and $Q(k) = b_0+b_1k + \cdots b_{q}k^{q}$ where $a_p\ne 0 \ne b_q$. Assume $Q(k)\ne 0$ for $m\le k$.
Then $$\begin{aligned} \lim_{k\to\infty}\frac{a_0+a_1k+\cdots+a_pk^p}{b_0+b_1+\cdots+b_{q}k^{q}} & = \lim_{k\to\infty}\frac{a_0/k^q+a_1/k^{q-1}+\cdots a_p/k^{q-p}}{ b_0/k^q+b_1/k^{q-1}+\cdots+b_q } \\ &= \frac{\lim_{k\to\infty}(a_0/k^q + a_1k^{q-1}+\cdots +a_p/k^{q-p})}{\lim_{k\to\infty}(b_0/k^q + b_1/k^{q-1}+\cdots+b_q)} \\ &= \frac{0+0+\cdots+0}{0+0+\cdots+0+b_q} \\ &= 0 \end{aligned}$$
If $\text{deg}(P) = \text{deg}(Q)$ then $\lim_{k\to\infty}R(k)$ is the ration of leading coefficients. Let $P(k) = a_0+a_1k+\cdots+a_pk^p$ and $Q(k)=b_0+b_1k+\cdots b_pk^p$ where $a_p\ne 0\ne b_p$. Assume $Q(k)\ne 0$ for $m\le k$.
Then $$\begin{aligned} \lim_{k\to\infty} \frac{a_0+a_1k+\cdots+a_pk^p}{b_0+b_1k+\cdots+b_pk^p} &= \lim_{k\to\infty}\frac{a_0/k^p+a_1/k^{p-1}+\cdots+a_p}{b_0+b_1k+\cdots+b_p} \\ &= \frac{\lim_{k\to\infty}(a_0/k^p+a_1/k^{p-1}+\cdots+a_p)}{\lim_{k\to\infty}(b_0+b_1k+\cdots+b_p)} \\ &= \frac{0+0+\cdots+0+a_p}{0+0+\cdots+0+b_p} \\ &= \frac{a_p}{b_p} \end{aligned}$$
