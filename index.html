Unit 1: Real Number Axioms

Lessons
  • Lesson 2: Groups
    1. Notation: $xy$, $e$, $x^{-1}$, $x/y$, $x^n$
    2. Definitions: identity element, inverse element, group, commutative
    3. Theorems:
      • Assoc. op. uniq. id. & inv.
      • Group. op. uniq. id. & inv.
      • Group inv. inv.
      • Group uniq. sol.
      • Group prod. inv.
      • Group exp. laws
  • Lesson 3: Rings and Fields
    1. Notation: $0$, $1$, $na$, $a^n$
    2. Definitions: distributes, ring, field
    3. Theorems:
      • Ring uniq. id. & inv.
      • Ring zero annihilates
      • Ring -1 negates
      • Field eqs. uniq. sols.
  • Lesson 4: Order
    1. Notation: $\preceq,\prec,\succeq,\succ$
    2. Defs: Partial order, partially ordered set, poset, commensurability, totality, total order, totally ordered set
    3. Thm: Poset prec. strict part. ord.
    4. Thm: Poset neg. prec.
  • Lesson 5: Bounds
    1. Notation: $UB_A, \max(A), \sup(A)$
    2. Defs: Upper bound, maximum, supremum, and duals (lower bound, minimum, infimum)
    3. Thm: Bound max. is sup.
    4. Thm: Bound tot. ord. finite set max.
    5. Thm: Bound inf. lefteq sup.
    6. Thm: Bound subset sup. ineq.
  • Lesson 6: Function Bounds
    1. Notation: $\text{Im}(f), \max(f),\sup(f),f\preceq g, f_{(\cdot, c)}, f_{(c,\cdot)}, \sup_{x\in X}f_{(x,\cdot)}, \sup_{y\in Y}f_{(\cdot,y)}, \sup_{x}\sup_y f$
    2. Defs: Image, function bound, function maximum, function supremum, partially applied function
    3. Thm: Fun. Bound Ineq. Sup. Ineq.
    4. Thm: Fun. Bound Double Sup.
  • Lesson 7: Ordered Fields
    1. Notation: $(a,b), (a,b], [a,b), [a,b],\infty, F^\ast, a+X, aX X+Y, XY, X^+, X^-, X^{\preceq 0}, X^{\succeq 0}$
    2. Def: Interval, bounded, closed, open, positive, negative, compatible with addition, compatible with multiplication, field of fractions
    3. Thm: Ord. field simult. comp.
    4. Thm: Ord. field strict prec. compat.
    5. Thm: Ord. field zero prec. one
    6. Thm: Ord. field prod. pos. neg.
    7. Thm: Ord. field integer powers
    8. Thm: Ord. field dual
    9. Thm: Ord. field alg. sup.
    10. Thm: Ord. field func. sum sup.
  • Lesson 0008: Completeness
    1. Notation: $\Bbb R,\Bbb Q,\le,\sqrt[n]x, x^{p/q}$
    2. Defs: Completeness (with respect to suprema), complete ordered field, real numbers, roots, rational exponents.
    3. Axiom: Real axiom.
    4. Thm: Real roots exist.
    5. Thm: Real pow. frac. indep. ord.
    6. Thm: Real rational pow. laws
